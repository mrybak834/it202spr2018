Answers to questions
  1. The Box Model is a styling property of every html element, consisting of a box structure that wraps around the content. The box consists of the padding, border, and margin, all of which can be manipulated.
  2. A block-level element is an html element that occupies the entire space of the parent container.
  3. A <form> is a block-level element
  4. An inline element is an element which only occupies the space within which it is defined, not taking up the whole container. Inline elements don't break content flow and don't create newlines.
  5. <span> is an inline element
  6. You can include css via importing an external stylesheet, or using an internal stylesheet (<style> defined in the <head> of the page). It is also possible to style elements in-line using <style>, but this is frowned upon.
  7. A selector designates which elements the css rule will apply to
  8. body { background: #0000FF; }

  9. img { width: 75%; }
