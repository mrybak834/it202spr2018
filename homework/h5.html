# Answer
1 You include Bootstrap along with requisite jQuery via CDN, using a link tag
2 A CDN is a Content Delivery Network: An online file host that caches resources for quick delivery to the user, improving page load speeds and saving server space
3 A CDN allows for faster page loads by caching frequently used resources, so they do not have to be re-downloaded
4 A breakpoint is a media query action that specifies the device screen size alteration at which to modify the layout of the content.
5 The Bootstrap breakpoints are: 576, 768, 992, and 1200px.
6 A fixed-width container changes it's max-width at every breakpoint, while a fluid-width container maintains a 100% width always.
7 The Bootstrap Grid is 12 columns
8 Rows counteract column padding, breaking up the grid horizontally and giving content a visual left alignment.
